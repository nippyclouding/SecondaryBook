## 0. 테스트 환경 준비

```
서버 URL   : http://localhost:8080
DB         : 테스트 DB (운영 DB 사용 금지)
Redis      : 로컬 또는 테스트 인스턴스
S3         : 테스트 버킷 또는 로컬 파일 스토리지
Toss API   : 테스트 키 (clientKey/secretKey)
이메일     : 실 수신 가능한 테스트 이메일 계정 2개 이상
테스트 계정: 일반회원 A, 일반회원 B, 관리자
```

**테스트 계정 구성**

| 역할 | 아이디 | 비고 |
|---|---|---|
| 일반회원 A (판매자) | testA | 판매글 보유, 독서모임 모임장 |
| 일반회원 B (구매자) | testB | 잔액 보유 |
| 관리자 | admin | 2단계 접근코드 필요 |
